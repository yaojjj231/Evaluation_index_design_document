感知SIL评测的定义
=======================

.. toctree::
   :maxdepth: 3


- 感知SIL评测，用于评测感知任务，关注于感知任务的完整表现。被评测的感知任务，其输入是连续的传感器数据序列（有时间维度），输出包含了经过后处理或者跟踪之后的数据，比如速度（可能比MIL多了时间维度）或者三维坐标（比MIL有更多的空间维度）。

- SIL评测是感知任务评测的必备评测手段。
  - 评测感知能力的方式：MIL 评测、SIL评测、实车评测
  - MIL评测：模型评测
  - SIL评测：感知任务 基于（连续帧）评测集的开环评测
  - 实车评测：感知任务、基于随机路测（或设计场景）的、加规控的闭环评测

- 通过SIL评测验证，关键指标有正向提升的感知版本，可认为是满足SIL评测的版本，可进入到实车闭环评测环节。
- 感知SIL评测不包括预测任务，只反映对已发生事件的观测。
- SIL评测当前阶段，以通用评测为主（暨：SIL评测场景库为正向场景库）

感知SIL评测项目，按项目实现、可分为「SIL评测指标体系」、「SIL评测场景库」、「SIL评测执行规范」、「SIL评测工具」四部分。

感知SIL评测指标体系
=======================

感知SIL评测，按感知任务进行划分，不同的任务具有对应的评测指标。感知任务可分为：车辆、行人、二轮车、车道线、静态障碍物、停车库位等。
随着感知开发的逐渐成熟，被评测的任务会发生新增。
